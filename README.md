## Overview

This code classifies the image into the type of animal.

## Softwares

* Recommended IDE: PyCharm 2021.2

## Libraries

* Numpy 1.21.2
* Matplotlib 3.5.0
* SkLearn 1.1.3
* PyTorch 1.12.1+cu116

## Programming Languages

* Python 3.8.12
